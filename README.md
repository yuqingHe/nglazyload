# 主要功能
* 使用loclazyload实现angular模块的懒加载
* browser-sync实时编译sass文件并刷新;
* 使用grunt - requireJs按模块打包angular;

## 开发者模式
命令行切换到当前根目录,
输入  npm install  安装依赖
输入  grunt
浏览器打开后http://localhost:3000/index.html;

## 发布模式
打开Gruntfile.js文件 把90行注释,92行反注释 

命令行切换到当前根目录,
输入  npm install  安装依赖
输入  grunt
浏览器打开后http://localhost:3000/index.html;
